Exam Management System

In Real Time System, Exam Scheduling is very difficult task for Organizer. There are many difficulties and complications in allocating Exam Center, Exam Hall, Exam Date and Time. To solve this problem we initiated this project.

This project is all about software for college, school as well as organization which conducts board exam for students. Exams are conducted at various departments and exam centers on different dates at particular time. Particular batch of students are allowed to attend the exam at particular center/exam hall.

Main task of our project is to reduce the work load of exam conductor of allocating Exam center, Exam hall, with fixing Exam Date and Time, and help to generate Exam hall tickets with exam related information like student's Exam Center, Hall Number, Exam Date and Time etc. After creating hall tickets send it to each student’s mail id individually, this task is very complex and time consuming for exam scheduler as well as required more man power to execute this. So, to make it easy and fast we develop smart and effective exam scheduler software.
